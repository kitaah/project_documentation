# PHP 8.3

## **General**

### **Install PHP-FPM 8.3 with packages (Ubuntu 22.04)**
```sh
sudo apt install php8.3-fpm php8.3-{bcmath,bz2,cli,common,curl,dev,dom,gd,mbstring,imagick,intl,imap,ldap,pgsql,soap,xml,xmlrpc,zip}
```

### **Check PHP version**
```sh
php -v
```

### **Check packages**
```sh
php -m
```
